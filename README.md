# medsight-ai

A research-oriented Python framework for medical image classification and
segmentation experiments. Focused on reproducible pipelines for radiology
datasets (chest X-ray, retinal fundus, dermatology).

## Status

Pre-alpha. Interfaces will change without notice.

## Disclaimer

This project is for research and educational use only. It is not a medical
device and must not be used to make clinical decisions.
